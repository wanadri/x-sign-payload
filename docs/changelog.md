## Version History Overview

### v1.0.0 (Latest)
**Initial Release**

**Added:**
- ✅ PHP Laravel support with middleware and artisan commands
- ✅ PHP CakePHP support with PSR-15 middleware
- ✅ PHP CodeIgniter 4 support with filters
- ✅ JavaScript/TypeScript SDK for Node.js and browsers
- ✅ HMAC-SHA256 and HMAC-SHA512 algorithms
- ✅ Configurable timestamp validation
- ✅ Constant-time signature verification
- ✅ Secure secret generation utilities
