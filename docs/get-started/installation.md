# Installation

Choose your platform below for installation instructions.

## PHP

```bash
composer require wanadri/x-sign-payload
```

## JavaScript / TypeScript

:::tabs
== npm

```bash
npm install x-sign-payload
```

== yarn

```bash
yarn add x-sign-payload
```

== pnpm

```bash
pnpm add x-sign-payload
```

:::
